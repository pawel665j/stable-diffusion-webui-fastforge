# 📦 Additional Repositories Used in Build

This project integrates several external repositories to enhance functionality and user experience.

---

## ✅ Mandatory Dependencies

These repositories are essential for FastForge to function properly:

| Repository | Description |
|------------|-------------|
| [salesforce/BLIP](https://github.com/salesforce/BLIP) | Vision-language model for image captioning |
| [lllyasviel/huggingface_guess](https://github.com/lllyasviel/huggingface_guess) | Utility for guessing HuggingFace model types |
| [AUTOMATIC1111/stable-diffusion-webui-assets](https://github.com/AUTOMATIC1111/stable-diffusion-webui-assets) | Assets used by Stable Diffusion WebUI |
| [lllyasviel/google_blockly_prototypes](https://github.com/lllyasviel/google_blockly_prototypes) | Blockly-based prototype tools |
